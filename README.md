## ✨ Overview

CineScope is a full-stack movie discovery platform that lets users explore trending movies, search the TMDB database, view detailed movie information, and maintain a personal watchlist.

The application uses secure JWT authentication with HTTP-only cookies and stores user watchlists in MongoDB.
